Brain Bin is a work in progress web app for saving and organizing important coding snippets and notes. Built with Next.js and React, it features a sidebar for navigation, a language selector, and a clean, modern UI.

Live Demo:
https://brainbin-2f3cf1d1k-cprecs-projects.vercel.app/

Features (so far)
Sidebar navigation
Language selection dropdown
Monaco code editor for writing and viewing code snippets
Simple, responsive layout
Status
This project is not finished yet. You can view the current UI and basic features, but there is no full authentication, note saving, or backend functionality implemented.
